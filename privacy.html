Privacy Policy

GPTandme — ChatGPT Query Counter

Last updated: 2026-07-06

Data collection

This extension does not collect, transmit, or share any personal information. It does not send data to any server.

Data storage

All data (daily prompt counts and a running total) is stored locally on your device using the Chrome Storage API. No data ever leaves your browser.

Permissions

  • storage — used to save your prompt counts locally.
  • host_permissions (chatgpt.com, chat.openai.com) — required so the content script can detect when you send a prompt on ChatGPT.

Third-party services

This extension does not integrate with any third-party analytics, advertising, or tracking services.

User control

You can reset your counts at any time using the "Reset today" or "Reset all" buttons. Uninstalling the extension removes all stored data.
