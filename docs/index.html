GPTandME

A local prompt counter for supported AI chat websites.

Install from the Chrome Web Store Privacy Policy Support

What It Does

GPTandME counts prompts you send on supported AI chat websites, shows usage totals in the toolbar badge and popup, and stores the data locally in Chrome storage on your device.

Features

  • Counts user-sent prompts on supported AI chat websites.
  • Shows today, week, month, last 24 hours, streak, total, sessions, and model breakdown when available.
  • Includes an optional in-page counter for today's count.
  • Supports CSV export and import for your local usage data.
  • Provides reset controls for today's count or all stored counts.

Supported Services

Supported services currently include ChatGPT, Claude, Gemini, and Perplexity. GPTandME is an independent tool and is not affiliated with, endorsed by, or sponsored by those services or their owners.
