GPTandME

A local prompt counter for supported AI chat websites.

Documentation for version 1.4.0.

Install from the Chrome Web Store Privacy Policy Support

What It Does

GPTandME counts prompts you send on supported AI chat websites, shows the combined daily count in the toolbar badge and combined plus per-service usage totals in the popup, and stores the data locally in Chrome storage on your device.

Features

  • Counts user-sent prompts on supported AI chat websites.
  • Shows today, week, month, last 24 hours, streak, total, sessions, and model breakdown when available.
  • Shows today's and all-time counts for ChatGPT, Claude, Gemini, Perplexity, and Grok.
  • Includes an optional in-page counter for today's count.
  • Downloads and restores complete JSON backups.
  • Exports date,provider,model,count CSV usage files and continues to import legacy GPTandME CSV files by merging their counts.
  • Provides reset controls for today's count or all stored counts.

Supported Services

Supported services currently include the HTTPS browser chat sites for ChatGPT, Claude, Gemini, Perplexity, and Grok. Grok counting is optional and begins only after you enable access to grok.com in the popup. Service counts, model labels when available, and other usage summaries stay on your device. GPTandME does not store prompt text or a page's full URL path.

GPTandME is browser-only. It does not count Claude Code, native desktop apps, command-line tools, or direct provider API usage outside the supported browser chat sites, and it cannot be enabled in Chrome Incognito mode. GPTandME is an independent tool and is not affiliated with, endorsed by, or sponsored by those services or their owners.
