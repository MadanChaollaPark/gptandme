Privacy Policy

GPTandME

Last updated: 2026-07-08

Data Collection

This extension does not collect, transmit, sell, or share personal information. It does not send data to any server and does not use analytics, advertising, or tracking services.

Data Storage

All usage data is stored locally on your device using the Chrome Storage API. This includes prompt counts, daily/weekly/monthly totals, model-count summaries when available, hourly/session statistics, non-sensitive diagnostics such as the last counted time/source and last supported host, and local preferences such as the in-page counter setting. No data leaves your browser unless you manually export a CSV file from the popup.

Extension Surfaces

Version 1.2.3 exposes local usage information through the extension badge, the extension popup, and an optional in-page widget/counter on supported chat hosts. The popup may show a local OpenAI API cost proxy based on prompt-count assumptions; it is not actual billing data. The widget is optional; the badge and popup remain the primary supported surfaces.

Supported Hosts

The extension only runs on supported AI chat hosts: chatgpt.com, chat.openai.com, claude.ai, gemini.google.com, perplexity.ai, and www.perplexity.ai. It does not run on OpenAI platform, API, billing, help, documentation, or unrelated web pages.

Permissions

  • storage is used to save your prompt counts locally.
  • webRequest is used locally for ChatGPT request detection so user-sent prompts can be counted more reliably.
  • activeTab is used only when you open the popup so it can label whether the current tab is a supported host.
  • alarms is used locally to refresh the badge count after date changes while the browser is running.
  • host permissions are limited to chatgpt.com, chat.openai.com, claude.ai, gemini.google.com, perplexity.ai, and www.perplexity.ai so the content script can detect user sends on supported chat sites.

Third-Party Services

This extension does not integrate with any third-party analytics, advertising, or tracking services.

Limited Use

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

User Control

You can reset your counts at any time using the "Reset today" or "Reset all" buttons in the popup. You can also export or import a local CSV manually. Uninstalling the extension removes the locally stored extension data from the browser profile.
