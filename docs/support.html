GPTandME Support

Support information for the GPTandME Chrome extension.

Contact

Email chaollapark@gmail.com or report a reproducible technical problem in the public issue tracker. Do not include prompt text, exported usage data, or conversation URLs.

Before Reporting an Issue

  • Confirm you are using version 1.5.0 or newer for ChatGPT thinking-time reporting, or version 1.4.0 or newer for Grok counting.
  • Confirm you are using Chrome/Chromium 111 or newer.
  • Confirm the current site is one of the supported hosts: ChatGPT, Claude, Gemini, Perplexity, or Grok.
  • For Grok, open the popup, enable optional Grok counting, accept access to grok.com, and reload any Grok tab that was already open.
  • Confirm the activity happened in the browser chat site. Claude Code, native desktop apps, command-line tools, and direct provider API clients are outside GPTandME's scope.
  • GPTandME cannot be enabled in Chrome Incognito mode, so private-window activity is not counted.
  • Reload the extension after updating it, then reload every already-open supported-site tab.
  • Send a new prompt after the extension is loaded; historical prompts are not backfilled.
  • Usage recorded before service attribution was introduced may appear under the provider "unknown" rather than being assigned to a service.
  • ChatGPT average and total thinking time appear only for new ChatGPT responses after you send a prompt when ChatGPT exposes a finalized provider-reported timing label; historical conversations and untimed prompts are not backfilled.
  • Reset today deletes today’s prompt and thinking-time aggregates and clears session aggregates and recent deduplication IDs, while preserving older daily, service, model, and timing totals.

What To Include

For help with GPTandME, include your browser version, extension version, supported website, whether the combined or per-service count is affected, whether ChatGPT thinking time is affected, and whether you sent the prompt with the keyboard or the send button. Do not include prompt text, response text, exported usage data, raw timing labels, screenshots of conversation content, or conversation URLs.

Backup and CSV Compatibility

A JSON backup preserves the complete local dataset, including ChatGPT timing aggregates when present; restoring it replaces the data currently stored. Version 1.3.0 introduced CSV exports using date,provider,model,count. CSV files remain prompt-count only and do not include thinking-time aggregates. CSV imports merge by adding counts, so importing the same file twice duplicates them. Legacy GPTandME CSV files that use date,model,count remain importable and are recorded with provider unknown.
